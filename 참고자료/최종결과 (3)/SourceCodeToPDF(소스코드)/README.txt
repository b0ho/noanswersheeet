SourceCodeToPDF 프로그램은...
이 프로그램은 JAVA, C, C++ 언어로 작성 된 파일들을 PDF 문서로 변환시켜주는 프로그램 입니다.

기본 사용 방법
1. 프로그램 로드 완료후 <변환설정 - 언어 선택>을 통해 변환 하려는 코드의 언어를 지정합니다.
2. <적용> 버튼을 누른 후 <소스파일 로드 - 파일 혹은 폴더> 선택 후 변환 하려는 코드 혹은 코드가 있는 폴더를 지정합니다.
3. <변환하기 - 변환> 버튼을 누른 뒤 변환이 완료 될 때 까지 기다립니다.
4. 모든 변환이 끝난 뒤 프로그램이 실행된 위치에 PDF 파일들이 생긴것을 확인할 수 있습니다.

변환 옵션
 > 라인 넘버 표시 : PDF 문서에 코드 부분에 줄 번호를 표시할 지 여부를 지정합니다.
 > 변환 방식 : PDF의 출력 옵션을 지정합니다
  >> 종합 변환 : 변환 하려는 폴더 내의 모든 소스코드를 한 개의 PDF 문서로 출력합니다.
  >> 패키지 변환 : 변환 하려는 폴더 내의 소스코드를 각각의 폴더별 PDF 문서로 출력합니다.
  >> 개별 변환 : 변환하려는 폴더 내의 소스코드를 각 파일당 1개의 PDF 문서로 출력합니다.
 > 탭 길이 : 문서의 탭 길이를 지정합니다.
 > 헤더 내용 : 문서 머릿말에 들어갈 내용을 지정합니다. (최대 40자)
 > 해더 위치 : 머릿말이 나타날 위치를 지정합니다.
 > 저장 경로 : PDF 문서가 생성될 위치를 지정합니다.
 > 파일 이름 설정 : 종합 변환에서 생성되는 PDF 문서의 이름을 지정합니다.
이 옵션 값들은 모두 저장됩니다.

서식 설정
 > 용지 설정 : 용지의 크기 및 상하좌우 여백을 지정합니다.
 > 폰트 설정 : 폰트의 종류, 폰트 크기, 폰트 색상, 스타일을 지정할 수 있습니다.
  >> 폰트 추가 방법 : 프로그램의 하위 폴더에 존재하는 fonts 폴더 내에 트루타입 글꼴 (TTF) 폰트를 삽입한 뒤 프로그램을 재시작 하면 폰트가 추가됩니다.
이 옵션 값들은 모두 저장됩니다.

저작권 설정
 > 저작권 옵션 사용 : 저작권 옵션의 사용 여부를 지정합니다.
 > 저작자 정보 표시 위치 : 저작자 정보의 표시 위치를 지정합니다.
 > 저작자 정보 : 저작자 정보를 지정합니다.
이 옵션 값들은 모두 저장됩니다.

보안 설정
 > 보안 설정 사용 : 보안 설정의 사용 여부를 지정합니다.
 > 복사 가능 여부 : 체크 되어있을 시 복사가 가능하며 체크 해제시 문서 내용이 복사되지 않습니다.
 > 비밀번호 : PDF 문서에 비밀번호를 지정할 수 있습니다.
이 옵션 값들은 저장되지 않으며 프로그램 재 시작시 초기화 됩니다.
